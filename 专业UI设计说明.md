# 专业UI设计说明

## 设计理念

作为专业UI设计师，我重新优化了界面设计，确保：

1. **高可读性**：所有文字在白色背景下清晰可见
2. **专业感**：符合企业级应用的设计标准
3. **层次清晰**：视觉层次分明，信息易于理解
4. **一致性**：统一的设计语言和交互模式

## 颜色系统

### 背景色系
```css
--bg-primary: #ffffff;        /* 纯白 - 主要卡片背景 */
--bg-secondary: #fafbfc;      /* 暖白 - 页面背景 */
--bg-tertiary: #f5f7fa;       /* 浅灰 - 次要背景 */
--bg-hover: #f0f2f5;          /* 悬停状态 */
```

### 文字色系（确保高对比度）
```css
--text-primary: #111827;      /* 深黑 - 主要文字 */
--text-secondary: #374151;    /* 深灰 - 次要文字 */
--text-muted: #6b7280;        /* 中灰 - 弱化文字 */
--text-placeholder: #9ca3af;  /* 浅灰 - 占位符 */
```

### 强调色系
```css
--accent-primary: #1e40af;    /* 专业深蓝 */
--accent-secondary: #1e3a8a;  /* 更深蓝 */
--accent-light: #3b82f6;      /* 亮蓝 */
--accent-bg: #eff6ff;         /* 蓝色背景 */
```

### 状态色系
```css
--success: #059669;           /* 成功 - 绿色 */
--success-bg: #d1fae5;        /* 成功背景 */
--warning: #d97706;           /* 警告 - 橙色 */
--warning-bg: #fef3c7;        /* 警告背景 */
--error: #dc2626;             /* 错误 - 红色 */
--error-bg: #fee2e2;          /* 错误背景 */
```

## 字体系统

### 字号层次
```css
/* 标题 */
h1: 2.25rem (36px) - 主标题
h2: 1.5rem (24px) - 次标题
h3: 1.125rem (18px) - 小标题
h4: 1rem (16px) - 最小标题

/* 正文 */
正文: 14px - 默认文字
小字: 12px - 辅助文字
最小字: 10px - 标签文字
```

### 字重
- 标题: 700 (bold)
- 次标题: 600 (semibold)
- 正文: 400 (regular)

## 组件优化

### 1. 头部导航
- 简洁的蓝白配色
- 图标和文字清晰可读
- 按钮使用深蓝背景

### 2. 输入区域
- 浅灰背景输入框
- 深色 placeholder 文字
- 蓝色焦点状态

### 3. 卡片组件
- 纯白背景
- 细微边框
- 悬停加深边框

### 4. 仪表盘
- 单一蓝色圆环
- 深色分数显示
- 简约评级标签

### 5. 进度条
- 浅灰背景
- 深蓝填充
- 细微圆角

### 6. 按钮
- 主按钮: 深蓝背景 + 白色文字
- 次要按钮: 浅灰背景 + 深色文字
- 悬停状态: 颜色加深

## 交互设计

### 悬停状态
- 卡片: 边框加深 + 阴影加深
- 按钮: 背景色加深
- 链接: 下划线 + 颜色变化

### 焦点状态
- 输入框: 蓝色边框 + 蓝色光晕
- 按钮: 蓝色边框环

### 激活状态
- 维度卡片: 蓝色边框环 + 背景色

## 响应式设计

### 移动端 (< 640px)
- 单列布局
- 字号适当缩小
- 间距适当调整

### 平板端 (640px - 1024px)
- 双列布局
- 标准字号
- 标准间距

### 桌面端 (> 1024px)
- 三列布局（维度卡片）
- 并排显示（雷达图+建议）
- 最大宽度限制

## 可访问性

1. **对比度**: 所有文字对比度符合 WCAG AA 标准
2. **焦点可见**: 所有交互元素都有清晰的焦点状态
3. **键盘导航**: 支持 Tab 键导航
4. **语义化**: 使用正确的 HTML 语义标签

## 专业细节

1. **圆角一致性**: 使用 4px/8px/12px/16px 系统
2. **阴影层次**: 从 xs 到 lg 四个层次
3. **边框精细**: 1px 细边框，避免粗重感
4. **文字清晰**: 使用系统字体，确保渲染质量

## 设计优势

1. **高可读性**: 所有文字在白色背景下清晰可见
2. **专业感**: 符合企业级应用的设计标准
3. **一致性**: 统一的设计语言和交互模式
4. **可维护性**: 系统化的设计变量
5. **可访问性**: 符合无障碍设计标准
