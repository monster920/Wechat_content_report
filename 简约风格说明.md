# 简约风格UI设计说明

## 设计理念

从"超高级炫酷UI"转变为"简约风格UI"，主要特点：

1. **干净的白色/浅灰色背景**
2. **深色文字（黑色/深灰）**
3. **单一强调色（深蓝色）**
4. **简洁的边框和阴影**
5. **移除渐变和动画效果**

## 颜色方案

### 背景颜色
- 主背景：`#f8f9fa` (浅灰)
- 卡片背景：`#ffffff` (纯白)
- 次要背景：`#f1f3f5` (浅灰)

### 文字颜色
- 主要文字：`#1f2937` (深灰)
- 次要文字：`#6b7280` (中灰)
- 弱化文字：`#9ca3af` (浅灰)

### 强调色
- 主强调：`#2563eb` (深蓝)
- 次强调：`#1d4ed8` (更深蓝)
- 悬停状态：`#1e40af` (最深蓝)

### 边框颜色
- 主边框：`#e5e7eb` (浅灰)
- 次边框：`#d1d5db` (中灰)

## 组件样式变化

### 1. 全局样式 (index.css)

**移除：**
- 动态背景动画
- 霓虹灯效果
- 渐变文字
- 毛玻璃模糊效果

**新增：**
- 简约卡片样式
- 简化的阴影系统
- 淡入动画

### 2. 卡片样式 (glass-card)

**之前：**
```css
background: rgba(20, 20, 35, 0.6);
backdrop-filter: blur(20px);
border: 1px solid rgba(139, 92, 246, 0.3);
border-radius: 20px;
```

**现在：**
```css
background-color: #ffffff;
border: 1px solid #e5e7eb;
border-radius: 12px;
box-shadow: 0 1px 2px 0 rgba(0, 0, 0, 0.05);
```

### 3. 标题样式

**之前：**
```css
background: linear-gradient(135deg, #8b5cf6 0%, #06b6d4 100%);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

**现在：**
```css
color: #1f2937;
```

### 4. 按钮样式

**之前：**
```css
background: linear-gradient(135deg, #8b5cf6 0%, #06b6d4 100%);
```

**现在：**
```css
background-color: #2563eb;
```

### 5. 组件具体变化

#### ReportHeader
- 移除渐变文字
- 标签使用简约边框样式
- 文字颜色改为深灰

#### ScoreGauge
- 移除霓虹灯效果
- 仪表盘使用单一蓝色
- 评级标签使用简约边框

#### DimensionCard
- 移除渐变背景
- 状态标签使用彩色背景（绿/蓝/橙）
- 激活状态使用蓝色边框环

#### RadarChart
- 移除渐变填充
- 使用单一蓝色填充
- 网格线使用浅灰色

#### ActionList
- 移除渐变背景
- 使用简约边框和阴影
- 复选框使用标准样式

#### ExportToolbar
- 移除渐变按钮
- 使用深蓝背景按钮
- 其他按钮使用简约边框

## 响应式设计

保持原有的响应式布局：
- 移动端：单列布局
- 平板端：双列布局
- 桌面端：三列布局（维度卡片）

## 交互效果

- 悬停状态：边框颜色变化 + 阴影加深
- 点击效果：轻微缩放过渡
- 放大覆盖：平滑的固定定位过渡

## 字体和排版

- 主字体：系统默认 sans-serif
- 字重：主要使用 600 (semibold)
- 行高：1.6 (段落)
- 字号层次清晰

## 优势

1. **可访问性更好**：高对比度文字
2. **加载更快**：移除复杂动画和渐变
3. **更专业**：符合企业级设计标准
4. **更易维护**：简化样式系统
5. **更好兼容性**：减少CSS特性依赖

## 使用建议

1. 保持简洁的布局结构
2. 使用一致的间距系统
3. 避免过多的装饰元素
4. 保持色彩的一致性
5. 确保文字可读性
