# 🚀 快速启动指南

## 前置要求
- Node.js 18+ 已安装
- 项目已构建（`npm run build`）

## 快速启动（3步）

### 1. 安装依赖
```bash
npm install express compression
```

### 2. 启动生产服务器
```bash
node start-production.js
```

### 3. 访问应用
打开浏览器访问：`http://localhost:3000`

---

## 其他启动方式

### 方式一：使用 Vite 预览（最快）
```bash
npm run preview
# 访问: http://localhost:4173
```

### 方式二：指定端口启动
```bash
PORT=8080 node start-production.js
# 访问: http://localhost:8080
```

### 方式三：后台运行（Linux/Mac）
```bash
nohup node start-production.js > server.log 2>&1 &
```

## 常见问题

### Q: 端口被占用怎么办？
A: 修改 `start-production.js` 中的 PORT 或使用环境变量：
```bash
PORT=8080 node start-production.js
```

### Q: 如何停止服务器？
A: 按 `Ctrl+C` 或使用命令：
```bash
pkill -f "node start-production.js"
```

### Q: 如何查看日志？
A: 服务器启动时会显示访问地址和状态信息
