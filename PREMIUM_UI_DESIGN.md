## 🌈 全新配色方案

### 主色调 - 紫蓝渐变
```

卡片背景: rgba(20, 20, 35, 0.6) (半透明深蓝)
强调色:

渐变: #8b5cf6 → #06b6d4 (紫到青)
```

## 🎬 动画效果

### 背景动画
```css
@keyframes gradientFlow {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```

### 浮动动画
```css
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}
```

### 发光动画
```css
@keyframes pulse-glow {
  0%, 100% { box-shadow: 0 0 20px rgba(139, 92, 246, 0.3); }
  50% { box-shadow: 0 0 40px rgba(139, 92, 246, 0.6), 
                    0 0 60px rgba(6, 182, 212, 0.4); }
}
```

## 🎯 设计特点总结

| 特性 | 描述 | 实现方式 |
|------|------|---------|
| **毛玻璃效果** | 半透明背景 + 背景模糊 | `backdrop-filter: blur(20px)` |
| **紫蓝渐变** | 从紫色到青色的渐变 | `linear-gradient(135deg, #8b5cf6, #06b6d4)` |
| **霓虹灯效果** | 文字发光和阴影 | `text-shadow` + `box-shadow` |
| **动态背景** | 缓慢流动的渐变背景 | CSS 动画 + 背景定位 |
| **悬停效果** | 卡片浮起 + 边框发光 | `transform: translateY()` + `box-shadow` |
| **平滑过渡** | 所有交互都有过渡动画 | `transition: all 0.3s ease` |

## 🚀 使用方法

1. **启动开发服务器**
   ```bash
   npm run dev
   ```

2. **查看效果**
   - 打开浏览器访问 `http://localhost:5175`
   - 点击"使用模拟数据"按钮查看完整效果

3. **体验交互**
   - 悬停在卡片上查看发光效果
   - 点击"查看详情"展开子项
   - 使用底部工具栏导出报告
