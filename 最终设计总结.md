## 专业设计特点

### 1. 颜色系统（确保可读性）

```css
/* 背景色 */
--bg-primary: #ffffff;        /* 纯白 */
--bg-secondary: #fafbfc;      /* 暖白 */
--bg-tertiary: #f5f7fa;       /* 浅灰 */

/* 文字色（高对比度） */
--text-primary: #111827;      /* 深黑 - 主要文字 */
--text-secondary: #374151;    /* 深灰 - 次要文字 */
--text-muted: #6b7280;        /* 中灰 - 弱化文字 */

/* 强调色 */
--accent-primary: #1e40af;    /* 专业深蓝 */
--accent-bg: #eff6ff;         /* 蓝色背景 */

/* 状态色 */
--success: #059669;           /* 成功绿 */
--warning: #d97706;           /* 警告橙 */
--error: #dc2626;             /* 错误红 */
```

### 2. 字体层次系统

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| h1 | 36px | 700 | 主标题 |
| h2 | 24px | 600 | 次标题 |
| h3 | 18px | 600 | 小标题 |
| 正文 | 14px | 400 | 默认文字 |
| 辅助 | 12px | 400 | 辅助文字 |
| 标签 | 10px | 500 | 标签文字 |

### 3. 卡片设计

**主卡片 (card-primary)**:
```css
background: #ffffff;
border: 1px solid #f3f4f6;
border-radius: 12px;
box-shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.05);
```

**特点**:
- 纯白背景确保文字清晰
- 细微边框增加层次感
- 轻微阴影提升立体感

## 交互设计

### 悬停状态
- 卡片：边框加深 + 阴影加深
- 按钮：背景色加深
- 链接：下划线 + 颜色变化

### 焦点状态
- 输入框：蓝色边框 + 蓝色光晕
- 按钮：蓝色边框环

### 激活状态
- 维度卡片：蓝色边框环 + 背景色

## 响应式设计

### 移动端 (< 640px)
- 单列布局
- 字号适当缩小
- 间距适当调整

### 平板端 (640px - 1024px)
- 双列布局
- 标准字号
- 标准间距

### 桌面端 (> 1024px)
- 三列布局（维度卡片）
- 并排显示（雷达图+建议）
- 最大宽度限制
